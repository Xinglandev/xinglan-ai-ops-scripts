# API 规范 v0.8（已废弃）

> **注意**: 本规范对应已下线的 v0.8 版本，仅作存档用途
> **停用日期**: 2024-06-30
> **替代版本**: v1.2+

## 基础信息
- **API 版本**: 0.8.4
- **协议**: HTTP/1.1
- **数据格式**: JSON, XML (已弃用)
- **认证方式**: Basic Auth, API Key (已弃用)
- **速率限制**: 100 请求/分钟/用户

## 过时的接口列表

### GET /api/v0.8/legacy/users
**描述**: 旧版用户查询接口（已替换为 v1.0）

**端点**: `GET https://api.company.com/api/v0.8/legacy/users`

**请求参数**:
- `format` (string, optional): 响应格式，可选值: `xml` | `json` (默认: `json`)
- `page` (integer, optional): 页码 (默认: 1)
- `page_size` (integer, optional): 每页数量，最大值 50 (默认: 20)
- `status` (string, optional): 用户状态，可选值: `active` | `inactive` | `suspended`
- `created_after` (string, optional): 创建时间过滤，格式: YYYY-MM-DD
